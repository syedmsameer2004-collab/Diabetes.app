
Diabetes Risk Assessment Web App

This is a professional Streamlit-based application that predicts diabetes risk using a trained Random Forest model.

Features

- Clinical-style user interface with patient health inputs
- Risk score and health indicators (Glucose, BMI, Blood Pressure)
- Explanation of each input parameter
- Feedback form for users

How to Run

1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Run the app:
   streamlit run app.py
4. Open the browser at:
   http://localhost:8501

Model Information

The model (diabetes_model.pkl) was trained using the PIMA Indians Diabetes dataset.
