Diabetes Risk Assessment Web App
Professional Streamlit application for diabetes risk prediction from a trained Random Forest model.

Features

- User interface comparable to clinical with patient health inputs

- Health indicators and risk score (Glucose, BMI, Blood Pressure)
- Description of each input parameter
- Feedback form for users
How to Run

1. Clone the repository

2. Install dependencies:
pip install -r requirements.txt
3. Run the application:
streamlit run app.py
4. Open the browser at:

http://localhost:8501

Model Information

This tool was created using a model trained on the PIMA Indians Diabetes dataset.
